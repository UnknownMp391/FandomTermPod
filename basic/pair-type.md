# 配对类型

*也可以称为 CP 类型*

## BG

**Boy × Girl**  
**男 x 女**

传统意义上的男女配对, 在同人创作中并不少见, 但由于主流作品本身多为 BG, 通常不被特别区分

- 多以原作设定为基础
- 同人创作中常作为其他 CP 的 "情敌方" 出现

**示例**: 

- 《你的名字》泷×三叶
- 《犬夜叉》犬夜叉×戈薇

## GB

**Girl × Boy**  
**女 x 男**

与 BG 相反, 由女性主导配对关系, 强调 **女强男弱、女攻男受** 的设定  
GB 通常并不以性别排列顺序来区分攻受, 而是直接取向主导结构

- 多用于逆性别常规印象的构造 (如: 御姐×软男) 
- 女攻设定受欢迎程度逐年上升
- 少部分场合也直接用 BG 表示, 差异仅在 "谁主导"

**示例**: 

- 《魔女宅急便》琪琪×蜻蜓
- 《Fate/Grand Order》玛修×咕哒男主

::: info
部分现实语境中, "4i (四爱)" 即 GB 的一种口语替代, 意指 "异性恋中的女攻男受"
:::

## BL

**Boy’s Love**  
**男 x 男**

**在同人创作中最常见的配对形式之一**, 以男性角色间的情感为核心展开, 涉及友情、暗恋、爱恋、结合等多种情感强度  
是 "耽美" 题材的主要表现形式

- 原作可为友情、对手、上下属, 二创中改为恋爱向
- 攻受分明, 或采用 [无差](/relation/no-role-difference) 设定
- 可衍生多种子分类 (年下、强强、年上、主仆、竹马等)

**示例**: 

- 《魔道祖师》蓝忘机×魏无羡
- 《鬼灭之刃》富冈义勇×炭治郎 (同人创作)

::: info
**原耽** (原创耽美): 非改编、非二次创作、世界观原创的 BL 作品 *例如《默读》《撒野》等*
:::

## GL

**Girl’s Love**  
**女 x 女**

女女之间的情感表达与恋爱发展, 也是同人圈重要创作方向之一, 现实语境中常称为“百合”

- 感情发展多偏细腻、内敛、情绪拉扯
- 常见反差设定如 "软妹×御姐" "病娇×阳光" 等

**示例**: 

- 《终将成为你》小糸侑×七海灯子
- 《魔法少女小圆》晓美焰×鹿目圆

::: info
现实中“百合”可指女性同性恋群体, 但在同人圈更多强调创作内容本身, 无现实身份指代
:::
