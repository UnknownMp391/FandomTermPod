# 公公 & 嬷嬷

::: info
因为 公公 / 嬷嬷 的解释过于分散, 本篇百科仅从我能搜索到的资料和我亲身经历认知为参考

或许有些讲的东西会与你的认知不相符, 欢迎友好提出自己的意见和看法
:::

**"公公** 与 **"嬷嬷"** 是同人圈和饭圈中常见的戏称, 最初源自宫廷戏角色 *(太监、公公、嬷嬷)*, 表达特定类型的 CP 粉

后来被泛化为 CP 圈另一种 **表示角色左右攻受立场倾向** 的戏称

## 本意: 宫廷角色式调侃  

### 公公

原意来自宫廷中伺候皇帝的太监

在同人/饭圈中，用来戏称那些 **狂热护主、唯主独尊** 的粉丝

特征:

- 护自家 "主子" (某角色或艺人) 胜过一切
- 不希望 "主子" 与他人绑定CP, 甚至拆CP
- 喜欢说 "主子高贵" "主子不该倒贴" 等
- 发言风格强硬霸道, 有时带有鄙视对家的意味

### 嬷嬷

原意来自古装剧中的老嬷嬷, 擅长操心与劝导

用来形容那些 **苦口婆心、心疼主子命运** 的粉丝

特征:

- 强调主子苦情人设
- 喜欢进行情感脑补和角色心理分析
- 风格偏唠叨, 感性, 有 "老妈妈粉" 气质

## 引申: 攻受立场的调侃用语 (泛化)

在 **耽美/BL CP 圈** 中, "公公"  "嬷嬷" 逐渐演变为对 **CP粉攻受立场倾向** 的戏称

### 公公 (攻粉)

指在某对 CP 中坚持 "我推是攻" 的粉丝

往往强调气场压制, 外貌身高, 主动性等 "攻的要素"

会反对角色 "被压" 或当受，甚至参与 "争攻大战”

> 例如: "怎么又让我们 A 当受了? 公公不同意!"

### 嬷嬷 (受粉)  

指坚持 "我推是受" 的粉丝

强调角色娇软, 受宠, 被疼爱的一面

喜欢为角色编织苦情受, 万人迷受等形象

> 例如: "我们 A 就是娇娇小可怜, 谁都得宠着他!”

::: warning
虽然 "公公"  "嬷嬷" 原为戏称或自嘲用语, 但在争吵中也可能带有贬义

在 CP 圈讨论中，这两个词常与 **"争攻争受" "控评" "拆逆" "泥塑"** 等现象挂钩, 有一定争议性

使用时请注意语境与语气, 避免激化矛盾
:::
