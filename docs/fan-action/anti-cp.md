# 拆家 & 逆家 (拆逆)

将本来的一对 CP A x B

**拆家**: 变为 **AC, CA, BC, CB**

**逆家**: 变为 **BA**

::: tip
拆逆情况通常发生在**官配 CP**身上

也称为拆官配, 逆官配
:::

## 经常被拆逆的 CP 的常见特征

- 互动模糊 / 双向皆通
- **疯批强受** / **病美人攻** 演绎风潮影响
- 攻受标签模糊 → 双箭头产出多
- 角色性格反差激烈, 谁当攻都合理
- 大粉产出带动饭圈分裂

## 举例

| 原作 | 常见正向CP | 常见逆向CP | 说明 |
| --- | --------- | --------- | --- |
| 《鬼灭之刃》 | 炭善 (炭治郎×善逸) | 善炭 | 两人互动有爱、年龄相仿, 性格互补, 攻受分歧大 |
| 《咒术回战》 | 五悠 (五条×悠仁) | 悠五 | 强强互动, 老师学生设定常被逆; 五条美人设也被 "受化" |
| 《名侦探柯南》 | 赤安 (赤井×安室) | 安赤 | 冷战 + 敌对设定本身就给逆拆提供了很多素材 |
| 《进击的巨人》 | 利艾 (利威尔×艾伦) | 艾利 | Levi 虽年长但身高低 + 清冷形象常被 "受化" |
| 《天官赐福》 | 花怜 (花城×谢怜) | 怜花 | 原作强攻设定清晰, 但也有怜攻党坚持谢怜 "道貌岸然疯批攻" 设定 |
| 《魔道祖师》 | 忘羡 (蓝忘机×魏无羡) | 羡忘 | 夷陵老祖时期魏无羡有很明显的攻属性 |
