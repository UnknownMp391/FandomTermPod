# 推

**"推"** 是一种来源于日本偶像文化的用语, 指的是 "我支持 / 喜欢的角色、人物、团体等"

现在, "推" 泛指任何被个人偏好所**认同、支持、喜爱的对象**, 具有强烈的**主观情感色彩**

## 含义与用法

 "推" 字本意为 "推荐", 但在二次元及同人文化中, 已演化为一种身份表达, 表示 "我最喜欢的那个"

### 常见表达：

- **我推**：我最喜欢的角色、CP、艺人等
- **推推 / 推一推**：表示支持、推荐 (更轻度) 
- **本命推**：最喜欢的那一个, 具有长期稳定性
- **墙头推**：一时喜欢的角色 (热情可能更换) 

---

## 推的对象

| 类型         | 举例说明                          |
|--------------|-----------------------------------|
| 虚拟角色     | 动漫、游戏中的角色, 如 "我推是安室透"  |
| 真人艺人     | 偶像、演员, 如 "我推是某团成员"        |
| CP (配对)     | 如 "我推的是赤安, 不吃逆"              |
| 团体/作品     | 如 "我推这个组合" 、 "我推这部动画"      |

::: info
有些 "推" 党会对 CP 吃逆、拆推的行为反感, 产生圈层摩擦 *(激推)*
:::

## 衍生用法

- **推人设 / 推设定**：指喜欢某个角色的设定而非角色本身  
- **推宣**：发布内容为表达对 "推" 的爱意和支持  
- **推榜 / 推票**：为 "推" 争取曝光、排名的行为, 常见于粉圈

## 示例

```text
我推是杰克船长, 十年如一日。
你也推炭治郎? 太巧了我也推他!

```
