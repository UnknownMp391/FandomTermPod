# 哨向

"哨向" 全称为：

> 哨兵（Sentinel）& 向导（Guide）

该设定起源于 1990 年代美国电视剧《The Sentinel》, 后来被同人圈发扬光大, 逐渐演变成一个独立世界观, 尤其在欧美与中文圈的二创中被广泛使用

一种基于精神力与感知能力划分的 **"双人能力体系"**, 强调 **精神链接与能力互补**

## 主题特征

- 精神依存关系: 体现互补性与强绑定
- 社会分化: 根据觉醒与能力等级分类
- 强情感羁绊: 有时带有灵魂伴侣、命定搭档设定
- 灵能世界观: 强调精神力、共鸣与情绪感知

## 哨兵

| 特性       | 描述                                         |
| -------- | --------------------------------------------- |
| **身体能力** | 力量、速度、感知极强 (听力、嗅觉、视力等超越常人) |
| **精神状态** | 容易精神过载、失控 (俗称 "暴哨")               |
| **依赖向导** | 向导能通过精神场域安抚他们、避免崩溃            |
| **职业形象** | 军人、特工、警察、战士、守护者等                |

### 精神暴走

哨兵若精神场过载、缺乏向导调节, 可能陷入"精神暴走" 状态, 攻击欲增强、敌我不分, 需向导及时安抚

部分设定中, 暴走哨兵的精神力还可能 "感染" 他人, 引发他人精神紊乱甚至短暂哨兵化

## 向导

| 特性        | 描述                             |
| ----------- | ------------------------------ |
| **精神能力** | 情绪共鸣、安抚暴哨、读取精神场      |
| **辅助作用** | 支撑哨兵的精神稳定, 提升战斗效率    |
| **常见设定** | 灵魂触碰、精神链接、共振通感        |
| **职业形象** | 心理医生、科研人员、指挥官、治疗者等 |

### 逆暴

在部分扩展设定中, 精神负担过重的向导也可能精神暴走, 称为 "逆暴"  
这类向导通常精神力等级极高, 一旦失控, 波及范围更广, 甚至可能反向镇压哨兵

## 配对

哨兵与向导之间通常会形成一种独特的精神链接, 俗称 **"配对"、"精神绑定"**  
可能包括：

- 精神结合: 向导与哨兵通过精神链接达成共生关系, 通常伴随强烈情感羁绊  
  可带来力量提升, 也可能造成依赖 ~~精神 do~~
- 精神领域: 或在梦境或精神场中交互
- 灵魂兽: 哨向可各自拥有动物形态的精神象征 (例如狼、猫头鹰、豹等)
> 哨兵与向导的精神体常以动物形象呈现, 既是精神力量的象征, 也能作为 "配对兼容性" 的具象表现  
> 精神体之间也能互动甚至战斗, 是精神链接强弱的直观体现

::: tip
有些类似于 ABO 设定中的 [标记](/background/worldview/abo#%E6%A0%87%E8%AE%B0)
:::

### 结合热

部分设定中, 未结合的哨兵 / 向导会周期性陷入生理渴求状态 *🚗梗高发区*

::: tip
 ABO 设定中的 [发情期](/background/worldview/abo#%E6%A0%87%E8%AE%B0)
:::

::: info
结合热并非所有哨向设定中都有, 但作为 A/B/O 等设定影响下的衍生情节常被使用

它通常表现为精神场过热、失控性渴、甚至诱发绑定标记, 是推动哨向关系升级的高强度剧情节点
:::

## 常见道具

- 精神抑制贴: 类似 ABO 中的 [抑制剂](/background/worldview/abo#%E6%8A%91%E5%88%B6%E5%89%82), 可以**一定程度**防止哨兵精神暴走

## 为什么哨向受欢迎?

*DeepSeek:*

- 张力十足: 哨兵的失控与向导的克制形成天然戏剧冲突
- CP兼容性强: 适配任何原作, 只需赋予角色相应属性
- 🚗梗丰富: 结合热、标记、精神交融等设定便于展开成人向创作
