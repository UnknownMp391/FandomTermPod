# ABO

**ABO** 是一种非常常见的衍生设定, 用于构建角色之间的性别架构和社会结构

源于欧美同人圈, 但在中文圈也极为流行, 特别是在 BL（男男）和 BG（男女）同人作品中

> 一种超越 男 / 女 的 **"第二性别"**

## Alpha

首领型、强者、支配者

绝大多数时候以 **攻** 的身份出现

| 特性       | 描述                                                        |
| -------- | ------------------------------------------------------------ |
| **生理特征** | 力气大、精力旺盛、信息素强烈                                  |
| **行为特征** | 占有欲强、保护欲强、易冲动                                    |
| **信息素**   | 味道浓烈、对 Omega 有极强影响力                               |
| **发情模式** | 有时设定中也有 "发情期", 或称为 "易感期", 容易受 Omega 信息素影响 |
| **结合能力** | 能对 Omega 进行**标记** 并使其怀孕 (生子设定中)                 |

::: info
某些设定中 Alpha 的信息素有等级区分

等级高的 Alpha 可以用信息素去压制更低级的 Alpha 或 Omega
:::

## Beta

中性 / 普通人

| 特性       | 描述                                                  |
| -------- | ------------------------------------------------------ |
| **生理特征** | 接近现实中的普通人, 通常无信息素或极其微弱                |
| **行为特征** | 情绪稳定、适应性强, 是社会的 "润滑剂"                    |
| **信息素**  | 无气味, 或极其稀薄, 几乎不影响他人                        |
| **发情模式** | 没有发情期, 也不会受他人信息素诱导                       |
| **结合能力** | 一般无法和 Alpha 或 Omega 结合, 或结合不稳定 (部分设定例外) |

::: info
某些设定中 Beta 与 Beta 也可以配对, 类似正常的性别关系
:::

## Omega

顺从、弱势者

大多数时候以 **受** 的身份出现 *也可反设定为 "强O"*

| 特性       | 描述                                                    |
| -------- | -------------------------------------------------------- |
| **生理特征** | 通常体型偏小、气味浓烈, 会周期性进入发情期                  |
| **行为特征** | 敏感、情绪化、易受 Alpha 吸引, 也常被描写为隐忍、细腻        |
| **信息素**  | 具有诱惑性, 容易吸引 Alpha；可能有专属味道                  |
| **发情期**  | 定期或因外因诱发( 如受 Alpha 接近), 会引发强烈生理 / 心理反应 |
| **结合能力** | 可与 Alpha 结合、被标记、怀孕 (生子设定常见)                |


## 总览对比

| 性别 | 信息素 | 发情期 | 能否怀孕 | 常见设定角色 |
| --- | ------ | ---- | -------- | --------- |
| Alpha | 强烈 | 部分设定<br/>*易感期* | 否 (攻方) | 总裁、军官、年上攻 |
| Beta  | 微弱/无 | 无 | 否 (多数设定) | 普通职员、老师、理性人 |
| Omega | 强烈 | 有 | 是 (受方)  | 小职员、学生、病弱受  |

## 信息素 & 标记

### 信息素

信息素是一种虚构的激素或气味信号, 用于角色间的吸引、辨认、影响等, 通常与发情期、占有欲、情绪相关

- 每个 Alpha 和 Omega 都有自己独特的信息素味道 (如柠檬味、烟味、木质香等)Om
- 信息素可以引发对方的生理反应 (如发情、情绪变化)
- 有些设定中, 信息素会用于识别 "命定伴侣"

### 标记

标记是指 Alpha 对 Omega 进行的一种结合行为, 具有 "绑定" 甚至 "永久占有" 的象征  
分为几种：

#### 临时标记

一次性或短暂的结合, 信息素覆盖一段时间

可以短期安抚 Omega 情绪

#### 永久标记

通过咬合腺体 / 交配等方式形成永久绑定, 具有强烈信息素连接

这种标记形成后不可去除, 在少数的设定中有 **洗标记** 的方法, 但通常对 Omega 身体有伤害, 并且实施困难

> 永久标记的重要性, 不可替代性  
> A & O 都需要深思熟虑才能进行

::: tip
没有标记的 Omega 会被其他 Alpha 认为 "未占有", 有更高的 "危险" 或 "吸引力"
:::

## 发情期 (易感期)

对于未被标记 (或未标记别人) 的 Omega (Alpha), 都会周期性的进入**发情期**

此时生理需求会变得旺盛, 对信息素非常敏感  
渴望标记

可以 **临时标记** 或者用 **抑制剂** 硬撑过去

::: info
Alpha 信息素可能导致 Omega 发情

发情的 Omega 可能会引诱 正常 Alpha 发情
:::

对于已经被标记 (或标记过别人) 的 Omega (Alpha), 一般分为两种设定

1. Alpha / Omega 可以 **自行控制** 发情期 / 易感期 ~~想生就发情~~
2. 仍会周期性发情, 但是因为有伴侣了也很好处理

## :syringe: 抑制剂

**抑制剂** 是一种用于抑制信息素释放或控制发情期反应的药物, 是非常常见的辅助设定道具

- 避免在公共场合信息素泄露
- 控制或延迟发情期
- 在特殊设定中, 也可用于暂时 "伪装"
- 减弱信息素对 Alpha 的诱惑力

::: info
一些同人设定中, 长期**大剂量**使用抑制剂会导致身体损伤  
包括但不限于

- 性腺萎缩 / 功能退化
- 体味 / 气息混乱
- 排异反应 *过敏 皮疹 呕吐 等*
:::
