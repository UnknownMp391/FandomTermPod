# 骨科

**"骨科"** 是中文同人圈常用的 **擦边亲属向 CP 代称**, 一般用于指代:

> **兄妹 / 姐弟 / 兄弟 / 父子 等血缘或名义上亲属关系** 的 CP

该词源于 "骨科医院" 这一现实概念, 在同人语境中为规避敏感词、调侃擦边设定而广泛使用

## 起源与演化

最早可追溯至网络流传的调侃: "你们这是骨科医院该看的病！" 

因涉及**乱伦 / 近亲设定**, 较为敏感, 故圈内以 "骨科" 作代指

后逐渐成为 CP 名称修饰词, 例如 "骨科兄妹"、 "骨科父子" 

::: info
该用法为中文语境特有, 日语 / 英语圈并无直接对等词
:::

| 类型             | 示例描述                     |
|------------------|------------------------------|
| 血缘亲属关系     | 亲兄妹、亲姐弟、父子、兄弟等 |
| 名义亲属         | 养子与养父、领养姐弟、义兄妹 |
| 伪骨科（非血缘） | 没有血缘但有 "名义关系" 设定   |

## 伪骨科

> **看似为亲属关系 (如兄妹、父子、姐弟等), 但实际并无血缘关系** 的 CP 设定

属于 "骨科" 设定的一个衍生变种, 用于规避伦理争议、降低背德感, 同时保留 "亲属设定" 的情节张力
