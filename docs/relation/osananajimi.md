# 幼驯染 (青梅竹马)

**幼驯染 (幼馴染 / おさななじみ)** 是一种在 ACG (动画, 漫画, 游戏), 小说, 同人文化中非常常见的角色关系设定, 指的是**从小一起长大, 感情深厚的角色**

## 幼驯染

来源: 日语词汇「幼馴染 (おさななじみ) 」

意思: 指从小就认识, 一起长大的朋友, 感情深厚

性别: **中性**, 可指男女, 男男, 女女

使用场景: 多见于日本 ACG, 轻小说, 同人设定等

特征: 相识时间久, 了解彼此, 有时发展为恋人

ACG 中常见形象
- 温柔贤惠的青梅女主
- 活泼爱吵闹的幼驯染竹马
- 被 "天降" 角色 "夺爱" 的 "童年败犬"

## 青梅竹马

- 来源: 中国古典成语, 出自《南史·谢庄传》
- 原意: 小女孩提青梅, 小男孩骑竹马, 一起玩耍
- 引申义: 从小一起长大的男女, 关系亲密
- 传统性别: **男女组合** (男是竹马, 女是青梅) 
- 使用场景: 文学, 现实语境广泛, 也常用于 ACG 翻译中替代“幼驯染”

::: info
现代网络语境中, **"青梅竹马" 往往泛指所有 "幼驯染" 关系**, 无论性别
:::

## 竹马竹马

网络衍生词, 指 **男男之间的幼驯染关系**

用法场景
- 常出现在腐圈, BL 同人作品中
- 表示两位男性角色从小认识, 情感特殊, 有耽美暗示或明确的 CP 含义

## 青梅青梅

网络衍生词, 指 **女女之间的幼驯染关系**

用法场景
- 多见于百合圈, GL 同人设定中
- 表达两个女角色青梅相识, 情感深厚, 有百合倾向

## 语义演变

在现代语境中 *(尤其是同人作品)*, "青梅竹马" 几乎成为 "幼驯染" 的替代, 即使不符合传统成语所限定的男女搭配, 泛化为包括男男, 女女在内的一切**从小认识并有亲密关系的设定**
