# 弯恋直文学

**"弯恋直"** 是一种常见的同人创作题材, 通常指:

> 弯的一方爱上了表面上是直的另一方, 由此产生一系列暗恋、克制、错位的情感纠葛

本质上是一种典型的“单箭头错位感情线”, 因其强烈的张力与情绪密度, 在各类 CP 文中经久不衰

## 结构定义

| 角色设定    | 描述                                   |
|------------|--------------------------------------- |
| 弯方 (主动) | 先产生情愫, 往往是暗恋、试探、受伤的一方     |
| 直方 (被动) | 表面上是“直”, 但通常会经历动摇、挣扎甚至反转 |

注意: "直方" 不一定是真直, 可能是伪直、[深柜](/relation/deep-in-closet)、未觉醒等设定

## 常见剧情模型

- 弯恋直单箭头: 主角暗恋好友/上司/队友, 对方毫无察觉
- 弯恋直反转线: 剧情推进中, "直方" 逐渐动摇甚至先告白
- BE经典线: 因“对方是直的”而选择放弃、错过
- 默默守护系: 爱而不得, 甘心做朋友直到终章
- 情感错位线: 你以为你在弯恋直, 其实对方早就喜欢你
