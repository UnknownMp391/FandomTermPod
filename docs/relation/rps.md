# RPS

**Real Person Slash**  
**真人配对 / 真人同人**

::: danger
本页面介绍的 **RPS** 基于 **现实存在的公众人物** 为基础进行的虚构创作

:warning: **重要提醒:**
- **请勿混淆虚构与现实**
- **请勿将圈内内容带入公共平台或本人面前**
- **请勿以 RPS 内容进行人身攻击、饭圈争斗、对号入座**

本页面内容仅供**术语记录与文化研究**, 不鼓励、宣传或扩散任何涉及**未经许可的真人演绎行为**

*若你对该类创作感到不适，请主动避雷*
:::

## 概述

RPS 是 "Real Person Slash" 的缩写, 意指以**现实中存在的真人** (如艺人、偶像、运动员、主播等) 为原型进行二次创作的配对作品创作内容多为虚构, 表现为 "角色化后的真人"

在中文语境中, RPS 通常也被称为:

- 真人同人
- 真人配对
- 真人衍生 (偏官方/平台术语)

## 主要特征

| 特征         | 描述 |
|--------------|------|
| **对象为真人** | 非虚构角色, 通常是公众人物 (例如明星、歌手、选秀选手等) |
| **创作内容为虚构** | 虽以真人为基础, 但故事设定、情节、关系皆为作者创作 |
| **角色化倾向明显** | 同人作品往往将真人 "当成角色" 进行演绎 |
| **情感主导** | 与普通 CP 同人类似, 以情感发展、互动描写为主 |
| **创作敏感度较高** | 涉及真实人物, 需特别注意隐私与尊重问题 |

## 常见类型

- **团内配对**: 偶像团体成员之间的配对最为常见, 如双人舞台、互动频繁等成为创作触发点
- **竞技向配对**: 运动员之间因合作或对抗关系被配对
- **娱乐圈衍生**: 演员、歌手、主持人等组合配对
- **主播 / 实况主向配对**: 游戏主播、虚拟主播 (部分为真声) 等

## 圈内术语

| 术语       | 含义 |
|------------|------|
| **嗑**     | 指嗑真人的互动或同人内容, 如 "嗑到了" "嗑疯了" |
| **代号化** | 用英文缩写或代称代替真实名字, 避免扩散或引起争议 |
| **避雷 / 打码** | 为避免涉及本人或外网传播, 对内容进行隐晦化处理 |

## 与虚构同人的区别

| 项目       | 虚构同人                | RPS (真人同人)         |
|------------|-------------------------|--------------------------|
| 对象       | 二次元角色 / 虚构人物   | **现实中存在的公众人物**     |
| 剧情设定   | 完全创作或基于原作世界观 | **虚构设定 + 真人名义**     |
| 道德敏感度 | 一般                 | **较高, 需谨慎处理**        |
| 扩散原则   | 可较自由发布           | **通常主张 "圈地自萌"**        |

## 创作规范建议

- **非当事人视角创作**: 不要用 "我就是他" 或 "仿冒口吻" 进行代入描写
- **避免现实诽谤**: 不要写涉及真实争议、丑闻或家庭关系的内容
- **保持架空化**: 鼓励创作架空 AU（如校园、ABO、现代职场等）, 避免现实贴脸
- **规避影响本人**: 不在公开平台刷屏, 不 @ 本人, 不在公共评论区发布

::: info
常见回避方式包括使用代号 (如 A×B)、缩写、反代、打码图等形式
:::

## 平台政策参考

不同平台对 RPS 内容的管控严格程度不同, 一些主流创作平台（如 LOFTER、AO3）允许有限创作, 但也常设有 "真人创作须谨慎" 的社区守则

部分社交平台 (如微博、小红书、豆瓣) 在风控时期会对涉及真人内容进行屏蔽或降权

## 争议与注意事项

RPS 虽然是圈内常见创作形式, 但因涉及真人, 常伴随争议:

- **不当传播影响本人形象**: 如被误传为现实中恋人
- **当事人拒绝**: 部分公众人物曾明确拒绝被配对
- **被主流误解**: 外部读者不理解 "角色化" 与 "现实" 的界限

因此, **RPS 是高敏创作, 应特别注意边界感与私密性**

## 小结

RPS 是同人创作的重要一环, 但不同于虚构角色同人, 它建立在现实人物基础上, 更需要创作者具备成熟的边界意识和圈内自律

::: warning
RPS 不是 "现实嗑 CP" , 而是一种以角色化为前提的二次创作形式
:::
