# 内容分级 (Content Rating) 

"内容分级" 是指对同人作品的内容进行分类标注, 以便读者在阅读前了解作品可能涉及的题材、尺度与接受门槛
 
该机制来源于欧美的影视 / 出版分级制度, 在同人圈中也被广泛沿用、演化

## 目的与意义

- 提供预警, 尊重不同读者的接受范围
- 保障未成年用户的阅读安全
- 明确界定作品尺度, 方便筛选和检索

## 常见分级体系

| 分级标签 | 含义                                               |
|----------|----------------------------------------------------|
| **G (General)**   | 普通级, 适合所有人阅读, 通常为友情向、日常向, 无敏感内容             |
| **PG (Parental Guidance)** | 普遍级, 但部分内容可能需父母指导, 例如轻微暴力、轻度情绪波动等    |
| **PG-13**          | 13 岁以上, 包含暗示性内容、轻度性描写或一定程度的暴力、心理描写等   |
| **R (Restricted)**| 限制级, 通常包含明确性描写 (🚗) 、暴力、黑暗、成人向情节等          |
| **NC-17**          | 强烈限制级, 仅限 17 岁及以上成人阅读, 内容包含露骨性描写或重口主题    |

::: tip
一些站点会使用 **K / T / M、E (Explicit)、Mature** 等不同标记形式, 但大体含义一致
:::

## 中文同人圈常见标法

| 中文标记     | 对应英文 | 说明                             |
|--------------|----------|----------------------------------|
| **普/清水**  | G        | 无性描写、无暴力, 纯情感/剧情向       |
| **轻车**     | PG-13    | 含有暗示性情节, 但不明确描写            |
| **开车/肉**  | R / NC-17 | 明确性描写, 可能含描写🚗或涩涩剧情     |
| **车速很快** | NC-17    | 强烈成人向, 快速进入露骨情节          |
| **强车/重口**| NC-17+   | 涉及特殊癖好、边缘题材或心理高压内容   |

::: info
除分级外, 许多创作者会附加特定预警标签以提醒读者内容可能引起不适, 如：

- **暴力 / 杀戮 / 自残**
- **强制 / 非自愿 / 控制欲**
- **精神创伤 / 心理疾病 / PTSD**
- **疼痛描写**
- **NTR / BE / 修罗场**
:::
