# 结局类型

用来表示故事的情感走向和完整度

## HE

*Happy Ending, 圆满结局*

故事以积极、幸福的方式收尾，角色达成目标或收获美好关系

**特点**

- 符合主流观众期待, 情感满足感强

  > 我可以接受悲惨的过程, 但是我不能接受悲惨的结局

  谁不想看好的, 幸福的结局呢

- 常用于浪漫喜剧、冒险故事或治愈向作品

**例子**

- 有情人终成眷属 (如《傲慢与偏见》)
- 反派被击败，世界恢复和平 (如《哈利·波特》最终战)

**变体**

- HE+ (超圆满结局): 所有伏笔回收，角色未来明确 (如大婚 & 婚后日常)
- 伪 HE: 表面美好但隐含隐患 (如《盗梦空间》陀螺未停)

## BE

*Bad Ending, 悲惨结局*

故事以悲伤、遗憾或失败告终, 角色可能死亡、分离或理想破灭

**特点**

- 冲击力强，易引发观众深刻共鸣
- 常见于现实主义、黑暗向或宿命论作品

**例子**

- 主角牺牲 (如《泰坦尼克号》杰克沉海)
- 努力仍无法改变结局 (如《萤火虫之墓》兄妹双亡)

**变体**

- BE 美学: 悲剧被艺术化, 如 "死在最美时刻" (如《斩·赤红之瞳》全员阵亡)
- 开放式 BE: 未直接展现悲剧, 但暗示绝望 (如《星际穿越》库珀独自漂流)

## OE

*Open Ending, 开放结局*

故事未明确交代结果, 留给观众自行解读

**特点**

- 保留悬念, 激发讨论
- 适合主题复杂或追求余韵的作品

**例子**

- 角色未来未卜 (如《盗梦空间》陀螺是否倒下)
- 关系未定性 (如《爱在黎明破晓前》是否再相遇)

**变体**

- OE偏 HE/BE: 虽未明说, 但线索倾向某一方 (如《寄生虫》儿子幻想买房)
- 循环 OE: 结局暗示故事重启 (如《恐怖游轮》无限轮回)
